# Todo App

Simple To Do list created from a tutorial. Features:

- Add tasks
- Mark done / toggle
- Edit tasks (double-click)
- Delete tasks
- Persistent storage via `localStorage`

To run:

Open [todo-app/index.html](todo-app/index.html) in your browser.

Optional: serve with a simple static server (recommended for some browsers):

```bash
# Python 3
python -m http.server 8000

# then open http://localhost:8000/todo-app/
```
